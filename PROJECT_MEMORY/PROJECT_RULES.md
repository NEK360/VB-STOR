# VB STORE — PROJECT RULES

## Главные правила
1. Никогда не ломать существующую архитектуру
2. Не создавать дубли компонентов
3. Все данные хранить в src/data/
4. Компоненты не содержат жёстко прописанный текст
5. TypeScript Strict везде
6. Все анимации через Framer Motion или GSAP
7. Smooth scroll через Lenis
8. Image Optimization через next/image
9. Lazy Loading везде
10. Адаптивность: Desktop, Laptop, Tablet, Mobile

## Архитектурные правила
- Данные → src/data/
- Компоненты → src/components/
- Хуки → src/hooks/
- Утилиты → src/utils/
- Типы → src/types/
- Сервисы → src/services/
- Конфигурация → src/config/

## Цветовая система
- Белый: #FFFFFF
- Черный: #000000
- Светло-серый: #F5F5F5
- Темно-серый: #1A1A1A
- Акцент: #E5E5E5
